# Tier 4 — Expert: Od buildera k profesionálním AI řešením

---

## Pro koho je tento tier

**Pro pokročilé buildery:** V Tier 3 jsi stavěl nástroje pro sebe. Teď se naučíš stavět profesionální řešení pro ostatní — MCP servery, testované a nasazené produkty, dokumentované a udržovatelné systémy.

**Pro vývojáře:** Už programuješ a chceš Claude integrovat profesionálně. MCP architektura, agentní vzory, testování, nasazení — tohle je tvůj tier.

**Pro OSVČ a technické leadery:** Chceš nabízet AI řešení klientům nebo nasadit Claude ve firmě na profesionální úrovni. Potřebuješ rozumět celému cyklu — od konceptu přes stavbu a testování po nasazení a údržbu.

### Tohle je pro tebe, pokud:

- Chceš stavět MCP servery — napojit Clauda na vlastní data a systémy
- Chceš rozumět agentním vzorům a pokročilé architektuře
- Chceš nasazovat řešení do produkce — ne jen prototypy
- Chceš stavět pro ostatní, ne jen pro sebe — dokumentace, onboarding, UX
- Říkáš si „chci z toho udělat produkt nebo službu"

### Tohle pro tebe není, pokud:

- Claude jsi nikdy nepoužil — začni s Tier 1 nebo 2
- Chceš Clauda jen používat, ne stavět infrastrukturu — Tier 2 nebo 3 stačí
- Setup dokument ti zabere víc než 5 hodin — pravděpodobně budeš mít víc z Tier 3, který tě na Tier 4 připraví systematicky

---

## Co potřebuješ před startem

Tier 4 nepotřebuje předchozí tiery — ale potřebuje znalosti a nástroje. **Setup dokument „Než začneš"** ti řekne přesně co, a u každého bodu co neumíš najdeš vysvětlení a mini úkol.

Stručně — potřebuješ:
- ☐ Claude Code na pokročilé úrovni (ne "zkusil jsem jednou")
- ☐ Funkční Claude API volání z Pythonu
- ☐ Bezpečnostní povědomí — prompt injection, obranné techniky
- ☐ Python nebo Node.js, VS Code, GitHub, základní git
- ☐ Strukturované prompty — XML tagy, system prompty

**Absolvoval jsi Tier 3?** Setup zabere 15 minut.
**Jsi vývojář, co zná Claude?** 1-2 hodiny.
**Jsi builder bez Tier 3?** 3-5 hodin, ale zvládneš to.
**Setup ti zabere víc než 5 hodin?** Zvaž Tier 3 — dá ti strukturovaný základ.

## Co budeš umět na konci

### MCP a architektura

- ☑ Rozumím MCP architektuře — protokol, servery, klienti
- ☑ Umím najít a použít existující MCP servery z registru
- ☑ Postavil jsem vlastní MCP server v Pythonu
- ☑ Umím implementovat agentní vzor do hloubky

### Profesionální kvalita

- ☑ Umím testovat a evaluovat kvalitu AI výstupů
- ☑ Umím nasadit řešení do produkce
- ☑ Rozumím monitoringu, bezpečnosti a škálování

### Stavba pro ostatní

- ☑ Umím z prototypu udělat produkt — dokumentace, onboarding, UX
- ☑ Mám Expert Capstone — profesionální projekt od návrhu po nasazení
- ☑ Mám portfolio projektů z celého kurzu

## Co dostaneš

### Formát

- **10 pracovních dní** (2 týdny, Po–Pá)
- **~2.5–3 hodiny denně**
- **Online hodina s lektorem** 1× týdně (Den 5 a Den 10)
- Po kurzu obsah + volitelné mise zůstávají přístupné

### Struktura

**Týden 1 — MCP a pokročilé vzory**
Pochopíš MCP architekturu, prozkoumáš ekosystém, postavíš vlastní server a implementuješ agentní vzor do hloubky. Na konci týdne máš funkční MCP server a agenta.

**Týden 2 — Od prototypu k produktu**
Naučíš se testovat, nasazovat a hlavně: stavět pro ostatní. Dokumentace, onboarding, UX — to, co z prototypu dělá profesionální řešení. Na konci máš Expert Capstone.

### Gamifikace

- **Questy** — denní checkpointy
- **Habits** — expert návyky
- **Mise** — volitelné expertní projekty
- **~2 800 XP** z povinného obsahu, **~5 000 XP** maximum s misemi

### Výstupy, které si odneseš

| Výstup | Co to je |
|---|---|
| MCP architektura mapa | Přehled protokolu, komponent, datových toků |
| Funkční MCP server | Vlastní server v Pythonu napojený na reálná data |
| Agentní implementace | Funkční agent s plánováním a tool use |
| Testovací framework | Sada testů a evaluačních kritérií |
| Produktová dokumentace | README, onboarding, provozní doc — pro uživatele, ne jen pro sebe |
| Expert Capstone | Kompletní profesionální projekt |
| Portfolio | Přehled všech projektů z Tier 1–4 |
