# Real world prompting - cesky prehled

Tento kurz navazuje na zaklady prompt engineeringu a ukazuje, jak promptovaci techniky pouzivat v komplexnejsich realnych scenarich.

Je urceny pro studenty, kteri uz prosli [Prompt Engineering Interactive Tutorial](../prompt_engineering_interactive_tutorial/README.cs.md) nebo maji zakladni zkusenost s promptovanim.

## Co se studenti nauci

- Jak prenest zakladni techniky do realnych promptu.
- Jak analyzovat slozitejsi zadani.
- Jak iterovat nad promptem.
- Jak stavet prompt pro specializovanou domenu.
- Jak premyslet o kvalite vystupu v praktickem kontextu.

## Doporuceni pro vyuku

Tento kurz je vhodny po zakladnim prompt engineeringu. Studenti by meli u kazde lekce sledovat nejen finalni prompt, ale hlavne proces, ktery k nemu vede.
