# Tool use - cesky prehled

Tento kurz uci, jak Claude pouziva externi nastroje. Tool use je zaklad pro agenty, strukturovane vystupy a workflow, kde model nema jen odpovidat textem, ale take volat funkce nebo pracovat s externimi systemy.

## Co se studenti nauci

- Co je tool use a proc je dulezity.
- Jak definovat jednoduchy nastroj.
- Jak pouzit tool use pro strukturovane vystupy.
- Jak vypada kompletni workflow s nastroji.
- Jak ridit vyber nastroje pres tool choice.
- Jak postavit chatbot s vice nastroji.

## Doporuceni pro vyuku

Tento kurz je vhodny po zakladech API a prompt engineeringu. Studenti by meli nejdrive pochopit, ze model nastroj sam nespousti v systemu. Model pouze pozada o volani nastroje a aplikacni kod toto volani provede.
