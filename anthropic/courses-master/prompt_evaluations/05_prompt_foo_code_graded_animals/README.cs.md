# Promptfoo code-graded evals: animals - cesky prehled

Tato lekce ukazuje zakladni pouziti `promptfoo` pro evaluace hodnocene kodem. Priklad pracuje s jednoduchou ulohou kolem zvirat.

## Spusteni

Nejdrive nastavte promennou prostredi `ANTHROPIC_API_KEY`.

Potom spustte:

```bash
promptfoo eval
```

Vysledky zobrazite prikazem:

```bash
promptfoo view
```

## Poznamka pro studenty

Sledujte hlavne to, jak jsou definovana testovaci data a jak kod rozhoduje, jestli vystup modelu splnil ocekavani.
