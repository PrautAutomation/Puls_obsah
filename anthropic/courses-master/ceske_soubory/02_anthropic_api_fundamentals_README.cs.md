# Anthropic API fundamentals - cesky prehled

Tento kurz je vstupni bod pro praci s Claude API a Anthropic SDK. Je vhodny pro studenty, kteri s Claude API jeste nezacali nebo potrebuji sjednotit zaklady.

## Co se studenti nauci

- Jak ziskat API klic.
- Jak poslat prvni jednoduche pozadavky.
- Jak funguje messages format.
- Jak porovnavat modely Claude.
- Jak nastavovat modelove parametry.
- Jak pracovat se streaming odpovedmi.
- Jak posilat obrazky a psat vision prompty.

## Lekce

1. [Getting an API key and making simple requests](./01_getting_started.ipynb)  
   Prvni nastaveni, API klic a jednoduche requesty.

2. [Working with the messages format](./02_messages_format.ipynb)  
   Struktura zprav, role, obsah a zakladni komunikace s modelem.

3. [Comparing capabilities and performance of the Claude model family](./03_models.ipynb)  
   Prehled modelu Claude a rozdily v rychlosti, cene a schopnostech.

4. [Understanding model parameters](./04_parameters.ipynb)  
   Parametry jako `temperature`, delka vystupu a dalsi nastaveni.

5. [Working with streaming responses](./05_Streaming.ipynb)  
   Postupne zpracovani odpovedi pri streamingu.

6. [Vision prompting](./06_vision.ipynb)  
   Prace s obrazky a multimodalnimi vstupy.

## Doporuceni pro vyuku

Tento kurz doporucujeme projit jako prvni. Studenti by meli po jeho dokonceni umet samostatne zavolat Claude API, upravit zakladni parametry a pochopit rozdil mezi textovym a multimodalnim vstupem.
