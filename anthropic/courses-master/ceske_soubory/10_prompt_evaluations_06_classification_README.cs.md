# Promptfoo code-graded classification - cesky prehled

Tato lekce ukazuje, jak pouzit `promptfoo` pro klasifikacni evaluace hodnocene kodem.

## Spusteni

Nejdrive nastavte promennou prostredi `ANTHROPIC_API_KEY`.

Potom spustte:

```bash
promptfoo eval
```

Vysledky zobrazite prikazem:

```bash
promptfoo view
```

## Poznamka pro studenty

U klasifikacnich evaluaci je dulezite mit jasne definovane spravne odpovedi. Vsimnete si, jak dataset, prompt a grader dohromady tvori opakovatelny test promptu.
