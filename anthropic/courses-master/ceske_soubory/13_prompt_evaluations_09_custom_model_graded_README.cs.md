# Custom model-graded promptfoo evals - cesky prehled

Tato lekce ukazuje vlastni model-graded evaluace v `promptfoo`. Kombinuje vyhody modeloveho hodnoceni s vlastni logikou a vlastnimi kriterii.

## Spusteni

Nejdrive nastavte promennou prostredi `ANTHROPIC_API_KEY`.

Potom spustte:

```bash
promptfoo eval
```

Vysledky zobrazite prikazem:

```bash
promptfoo view
```

## Poznamka pro studenty

Tento pristup je vhodny pro slozitejsi evaluace, kde potrebujete posoudit kvalitu, uplnost nebo dodrzeni pravidel, ale zaroven chcete mit nad hodnoticim procesem vetsi kontrolu.
