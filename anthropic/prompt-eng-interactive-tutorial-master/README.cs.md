# Prompt Engineering Interactive Tutorial - cesky prehled

Tento kurz je podrobny interaktivni uvod do prompt engineeringu pro Claude. Je postaveny jako postupny kurz s lekcemi, cvicenimi a prostorem pro experimentovani.

## Cil kurzu

Po dokonceni by studenti meli umet:

- navrhnout zakladni strukturu dobreho promptu,
- rozpoznat bezne zpusoby selhani promptu,
- pouzit nejdulezitejsi techniky, ktere resi vetsinu praktickych problemu,
- lepe chapat silne a slabe stranky Claude,
- stavet prompty od nuly pro bezne pripady pouziti.

## Struktura kurzu

Kurz obsahuje 9 kapitol s cvicenimi a dodatek s pokrocilejsimi metodami. Je navrzeny tak, aby se prochazel v poradi kapitol.

### Zacatecnik

1. `01_Basic_Prompt_Structure.ipynb` - zakladni struktura promptu.
2. `02_Being_Clear_and_Direct.ipynb` - jasne a prime instrukce.
3. `03_Assigning_Roles_Role_Prompting.ipynb` - role prompting.

### Stredne pokrocily

4. `04_Separating_Data_and_Instructions.ipynb` - oddeleni dat a instrukci.
5. `05_Formatting_Output_and_Speaking_for_Claude.ipynb` - format vystupu a predvyplneni odpovedi.
6. `06_Precognition_Thinking_Step_by_Step.ipynb` - premysleni krok za krokem.
7. `07_Using_Examples_Few-Shot_Prompting.ipynb` - pouziti prikladu a few-shot prompting.

### Pokrocily

8. `08_Avoiding_Hallucinations.ipynb` - omezovani halucinaci.
9. `09_Complex_Prompts_from_Scratch.ipynb` - stavba slozitych promptu pro realne pripady.

### Dodatek

- chaining prompts,
- tool use,
- search and retrieval.

## Varianty kurzu

V teto slozce je varianta `Anthropic 1P`, ktera pouziva primo Anthropic API. Ve slozce `AmazonBedrock` je varianta pro Amazon Bedrock.

## Doporuceni pro studenty

Kurz nepreskakujte. Kapitoly na sebe navazuji a pozdejsi cviceni predpokladaji, ze studenti uz umi zakladni strukturu promptu, oddeleni instrukci od dat a praci s priklady.

U kazde lekce je dulezite nejen precist vysvetleni, ale opravdu menit prompt a sledovat, jak se meni odpoved Claude.
