# Prompt Engineering Interactive Tutorial - Bedrock Edition - cesky prehled

Toto je varianta kurzu Prompt Engineering Interactive Tutorial pro Amazon Bedrock. Obsahove se zameruje na stejne promptovaci techniky jako hlavni kurz, ale priklady jsou prizpusobene prostredi Bedrock.

## Cil kurzu

Po dokonceni by studenti meli umet:

- navrhnout zakladni strukturu dobreho promptu,
- rozpoznat bezne chyby a selhani promptu,
- pouzit prakticke techniky pro zlepseni odpovedi,
- chapat silne a slabe stranky Claude,
- stavet prompty od nuly pro bezne pripady pouziti.

## Struktura kurzu

Kurz obsahuje 9 kapitol s cvicenimi a pokrocily dodatek.

### Zacatecnik

- Basic Prompt Structure
- Being Clear and Direct
- Assigning Roles

### Stredne pokrocily

- Separating Data from Instructions
- Formatting Output and Speaking for Claude
- Precognition / Thinking Step by Step
- Using Examples / Few-Shot Prompting

### Pokrocily

- Avoiding Hallucinations
- Complex Prompts from Scratch

### Dodatek

- Chaining Prompts
- Tool Use
- Empirical Performance Evaluations
- Search and Retrieval

## Kdy pouzit tuto variantu

Pouzijte ji, pokud studenti pracuji s Claude pres Amazon Bedrock. Pokud pouzivaji primo Anthropic API, je vhodnejsi hlavni varianta v `prompt_engineering_interactive_tutorial/Anthropic 1P`.

## Doporuceni pro vyuku

Nejdrive vysvetlete rozdil mezi Anthropic API a Amazon Bedrock. Promptovaci principy jsou stejne, ale autentizace, klient a nektere technicke detaily volani se lisi.
