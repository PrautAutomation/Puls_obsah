# Stav ceske verze prompt engineering tutorialu

Tento soubor popisuje samostatnou ceskou vrstvu pro `prompt-eng-interactive-tutorial-master`.

## Hotovo

- Hlavni cesky rozcestnik `README.cs.md`.
- Cesky prehled `AmazonBedrock/README.cs.md`.
- Ceske pokyny `AmazonBedrock/CONTRIBUTING.cs.md`.
- Ceske notebooky pro `Anthropic 1P` vcetne kapitol 00 az 10.3.
- Ceske notebooky pro `AmazonBedrock/anthropic` vcetne kapitol 00 az 10_4.
- Ceske notebooky pro `AmazonBedrock/boto3` vcetne kapitol 00 az 10_4.
- Vlastni slozka `ceske_soubory` obsahuje ploche kopie vsech ceskych materialu z teto slozky.

## Pravidla prekladu

- Student-facing text je cesky.
- Kod, nazvy funkci, promenne, API parametry, model IDs, prikazy a konfiguracni hodnoty zustavaji zachovane.
- U notebooku jsou prelozene hlavne Markdown casti; kodove bunky zustavaji funkcni.
- Pokud je anglicky prompt soucasti cviceni, zustava jako spustitelny material a cesky text vysvetluje jeho ucel.

## Kontrola

Czech notebooky byly zkontrolovane jako platny JSON a kodove bunky zustavaji shodne s originaly.
