# Tier 2 — Power: Od uživatele k power userovi

---

## Pro koho je tento tier

**Pro zaměstnance:** Claude už znáš — občas napíšeš e-mail, shrneš dokument, zeptáš se na radu. Ale víš, že z něj dostáváš zlomek toho, co umí. Všechno děláš v jednom chatu, prompty píšeš z hlavy, a o nástrojích jako Claude v Chrome nebo Claude Code jsi jen slyšel.

**Pro OSVČ a freelancery:** Chceš Clauda zapojit do svého workflow naplno. Nechceš jen občas chatovat — chceš systém, který ti šetří hodiny každý týden. Potřebuješ vědět, které nástroje existují a který se hodí na co.

### Tohle je pro tebe, pokud:

- Claude používáš, ale nemáš v tom systém
- Prompty píšeš „jak to přijde" a výsledky jsou nevyrovnané
- Nikdy jsi nepracoval s XML strukturou nebo system prompty
- Nezkoušel jsi Extended Thinking, Claude v Chrome, Cowork, Excel nebo Claude Code
- Chceš z příležitostného uživatele přejít na power usera

### Tohle pro tebe není, pokud:

- Claude jsi nikdy nepoužil — začni s Tier 1 (Foundation) nebo projdi Setup dokument
- Už píšeš pokročilé prompty, řetězíš workflow a denně používáš víc nástrojů — podívej se na Tier 3 (Advanced)

---

## Co potřebuješ před startem

Před Dnem 1 projdi **Setup dokument „Než začneš"** a odškrtni všechny body:

- ☐ Účet na claude.ai (ideálně Pro plán)
- ☐ Orientace v rozhraní
- ☐ Alespoň jeden Project s instrukcemi
- ☐ Memory s alespoň 3 fakty
- ☐ Zkušenost s uploadem souborů
- ☐ Základní znalosti: iterace, bezpečnost, anatomie promptu

Pokud něco chybí, Setup dokument tě provede — zabere max 2 hodiny.

---

## Co budeš umět na konci

### Promptování a myšlení

- ☑ Umím psát strukturované prompty s XML tagy a system prompty
- ☑ Umím rozložit složitý úkol na vícekrokový workflow (prompt chaining)
- ☑ Vím, kdy zapnout Extended Thinking a co mi to dá
- ☑ Mám workflow knihovnu s šablonami pro opakované složité úkoly

### Nástroje

- ☑ Umím používat Claude v Chrome pro úkoly v prohlížeči
- ☑ Umím používat Cowork pro automatizaci souborů a úkolů
- ☑ Vyzkoušel jsem Claude v Excelu a PowerPointu na reálných datech
- ☑ Vyzkoušel jsem Claude Code — vím co to je, k čemu to je, a jestli to chci dál rozvíjet
- ☑ Mám osobní nástrojovou mapu — vím, který nástroj na co použít

### Systém

- ☑ Mám Power Project s pokročilými instrukcemi a XML strukturou
- ☑ Mám Capstone výstup — reálný pracovní deliverable vytvořený pokročilými technikami

## Co dostaneš

### Formát

- **10 pracovních dní** (2 týdny, Po–Pá)
- **~2.5–3 hodiny denně**
- **Online hodina s lektorem** 1× týdně (Den 5 a Den 10)
- Po kurzu obsah + volitelné mise zůstávají přístupné

### Struktura

**Týden 1 — Pokročilé promptování**
Naučíš se psát prompty tak, aby Claude dělal přesně to, co chceš, napoprvé. System prompty, XML strukturování, prompt chaining, Extended Thinking. Na konci týdne máš Power Project a workflow knihovnu.

**Týden 2 — Nástroje v praxi**
Poznáš všechny hlavní nástroje: Claude v Chrome, Cowork, Excel, PowerPoint, Claude Code. Každý si vyzkoušíš na reálném úkolu. Na konci týdne víš, co ti sedí, a máš nástrojovou mapu pro svou práci.

### Gamifikace

- **Questy** — denní checkpointy, sbírej XP za splněné úkoly
- **Habits** — nové návyky, které si postupně přidáváš
- **Mise** — volitelné větší projekty pro ty, kdo chtějí víc (šéf může požadovat)
- **~3 000 XP** z povinného obsahu, **~5 000 XP** maximum s misemi

### Výstupy, které si odneseš

| Výstup | Co to je |
|---|---|
| Power Project | Project s XML system promptem a pokročilými instrukcemi |
| Workflow knihovna | 3+ vícekrokové šablony pro složité úkoly |
| Thinking cheat sheet | Přehled: kdy Extended Thinking ano, kdy ne |
| Nástrojová mapa | Osobní přehled: který nástroj na co používám |
| Capstone | Reálný pracovní výstup vytvořený pokročilými technikami + nástrojem |

## Návaznost

| Tier | Úroveň | Kam tě dostane |
|---|---|---|
| Tier 1 — Foundation | Začátečník → Uživatel | Reflex používat Clauda, funkční systém |
| **Tier 2 — Power** | **Uživatel → Power user** | **Pokročilé prompty, všechny nástroje, workflow** |
| Tier 3 — Advanced | Power user → Pokročilý | Automatizace, API, bezpečnost, integrace |
| Tier 4 — Expert | Pokročilý → Expert | Vlastní nástroje, MCP servery, produkční nasazení |
