# Témata — Den 4: Slash Commands a Skills

## Core

- příkazy pro opakované builder úkoly,
- skill jako zabalené workflow nebo odbornost,
- testování, jestli je s pomocí skillu výstup opravdu lepší,
- dokumentace pro sdílení s kolegou.

## Developer spine

- `.claude/commands/` a lokální vs globální commandy,
- `SKILL.md`, frontmatter a invocation,
- placeholders jako `$ARGUMENTS`, `$SELECTION` a `$FILE`,
- kdy stačí slash command a kdy má smysl přejít na skill,
- verzování a sdílení přes git.
