# Tier 3 — Advanced: Od power usera k builderovi

---

## Pro koho je tento tier

**Pro zaměstnance:** Claude používáš denně, máš systém, zvládáš pokročilé prompty a nástroje. Teď chceš víc — automatizovat procesy, stavět vlastní nástroje, napojit Clauda na systémy, které v práci používáš.

**Pro OSVČ a freelancery:** Chceš s Claudem tvořit — ne jen chatovat, ale stavět. Vlastní automatizace, skripty, webové stránky, aplikace. Nemusíš být programátor — Claude Code píše kód za tebe, ty říkáš co chceš.

### Tohle je pro tebe, pokud:

- Claude Code jsi zkusil a chceš jít hlouběji
- Chceš stavět vlastní nástroje a automatizace
- Zajímá tě, jak Claude funguje „pod kapotou" (API)
- Chceš Clauda napojit na VS Code, Slack, GitHub nebo jiné nástroje
- Chceš rozumět bezpečnosti — co se může pokazit a jak se bránit
- Říkáš si „mám nápad na nástroj/aplikaci, ale neumím programovat"

### Tohle pro tebe není, pokud:

- Claude jsi nikdy nepoužil — začni s Tier 1 (Foundation)
- Claude používáš, ale neznáš pokročilé prompty a nástroje — začni s Tier 2 (Power)
- Jsi zkušený vývojář a chceš MCP servery a produkční infrastrukturu — podívej se na Tier 4 (Expert)

## Co potřebuješ před startem

Před Dnem 1 projdi **Setup dokument „Než začneš"** a odškrtni všechny body:

- ☐ Účet na claude.ai (Pro plán nebo vyšší)
- ☐ Funkční Power Project s pokročilými instrukcemi
- ☐ Claude Code nainstalovaný a funkční
- ☐ Základní orientace v terminálu
- ☐ VS Code nainstalovaný (doporučeno)
- ☐ Základní zkušenost s Claude Code
- ☐ Pokročilé promptování (XML tagy, prompt chaining, Extended Thinking)

## Co budeš umět na konci

### Bezpečnost

- ☑ Rozumím prompt injection a vím, jak se bránit
- ☑ Umím posoudit bezpečnostní rizika svých promptů a workflow

### Stavění

- ☑ Umím v Claude Code stavět reálné nástroje krok za krokem
- ☑ Postavil jsem funkční nástroj, který používám v práci
- ☑ Umím vytvořit vlastní slash commands a skills
- ☑ Umím s Claudem prototypovat vizuálně — weby, dashboardy, landing pages

### Integrace

- ☑ Umím používat Clauda ve VS Code
- ☑ Umím napojit Clauda na Slack a/nebo GitHub Actions
- ☑ Rozumím, co je API, a udělal jsem první volání
- ☑ Vím, co API odemyká a jestli to chci rozvíjet dál

### Systém

- ☑ Mám Builder Project s pokročilým setupem
- ☑ Mám security cheat sheet
- ☑ Mám funkční integraci s alespoň jedním externím nástrojem
- ☑ Mám Builder Capstone — komplexní projekt kombinující dovednosti

## Co dostaneš

### Formát

- **10 pracovních dní** (2 týdny, Po–Pá)
- **~2.5–3 hodiny denně**
- **Online hodina s lektorem** 1× týdně (Den 5 a Den 10)
- Po kurzu obsah + volitelné mise zůstávají přístupné

### Struktura

**Týden 1 — Builder foundations**
Pochopíš bezpečnost, ovládneš Claude Code na pokročilé úrovni, postavíš svůj první reálný nástroj a naučíš se customizovat Clauda pomocí skills a slash commands.

**Týden 2 — Design, integrace, API**
Naučíš se prototypovat vizuálně s Claudem, napojíš ho na VS Code, Slack a GitHub, a poprvé zavoláš API. Na konci máš komplexní Builder Capstone.

### Gamifikace

- **Questy** — denní checkpointy, sbírej XP za splněné úkoly
- **Habits** — nové návyky pro buildery
- **Mise** — volitelné stavitelské projekty
- **~2 800 XP** z povinného obsahu, **~5 000 XP** maximum s misemi

### Výstupy, které si odneseš

| Výstup | Co to je |
|---|---|
| Security cheat sheet | Přehled rizik a obran pro tvé workflow |
| Můj nástroj | Funkční aplikace/skript postavený v Claude Code |
| Skills & Commands | Vlastní slash commands a skills |
| Design prototyp | Vizuální výstup — web, dashboard, landing page |
| Funkční integrace | Napojení na IDE, Slack, nebo GitHub |
| API first call | Úspěšné první volání Claude API |
| Builder Capstone | Komplexní projekt kombinující všechny dovednosti |
