# Témata — Den 1: Bezpečnost a prompt injection

## Core

- přímá injekce, nepřímá injekce a sociální inženýrství,
- zkoušení útoků na vlastní Project,
- obrana přes pravidla, odmítnutí a validaci výstupu,
- bezpečnostní reflex před sdílením nástroje s ostatními.

## Developer spine

- untrusted input a práce s cizími daty,
- oddělení instrukcí a dat přes strukturu promptu,
- principle of least privilege,
- output validation před dalším zpracováním nebo odesláním do toolu,
- hranice mezi system promptem, user inputem a tool vrstvou.
