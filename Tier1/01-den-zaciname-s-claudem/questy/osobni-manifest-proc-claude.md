# Quest: Osobní manifest Proč Claude

| | |
|---|---|
| **Typ** | Quest (menší úkol) |
| **Oblast** | Orientace |
| **Priorita** | vysoká |
| **Odměna** | +60 XP |
| **Kapitola** | 1 — Orientace: Svět Clauda a AI Fluency |

## Co udělat

Napiš s pomocí Clauda krátký osobní manifest na téma „Proč chci používat Claude".

1. Otevři nový chat.
2. Napiš Claudovi 2 až 3 konkrétní důvody, proč ti má pomáhat.
3. Nech si navrhnout verzi manifestu.
4. Uprav ji tak, aby zněla jako ty.

## Kritérium splnění

Máš hotový text o délce alespoň 150 slov, který obsahuje alespoň 2 konkrétní situace z tvého života nebo práce.
